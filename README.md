# loanword-cn
为网络流行外来词、文化空缺词、无官方中文翻译的英文缩写，提供统一、优雅、可落地的中文译名。

## 项目目标
互联网上大量英文单词、缩写、外来概念长期**无公认中文翻译**，导致交流混乱、表意不准。
本项目致力于：
- 收集无正经中文翻译的外来词
- 给出精准、简洁、符合中文语感的官方译名
- 形成开源、可共建、可引用的中文外来词标准词典

## 文件结构
- `words.json`：核心词条库，可程序读取
- `docs/`：详细说明与使用案例

## 参与贡献
- 提交 Issue 提议新词
- 提交 PR 完善翻译与解释
- 欢迎在文章、产品、文档中引用本项目译名
